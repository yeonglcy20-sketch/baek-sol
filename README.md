# 백솔 업보리스트

Google Spreadsheet와 연동되는 GitHub Pages용 업보리스트입니다.

## 적용 내용
- 제목: 백솔의 달콤살벌 업보 기록소
- 분홍머리 여자 캐릭터 백솔 콘셉트의 핑크 테마
- 프로필 카드, 통계 카드, 정렬 드롭다운, 워터마크 제거
- 총합/토탈 갯수 표시 제거
- 업보 항목 1열 구조
- 0 또는 빈칸은 자동 숨김

## 사용 방법
1. 압축을 풀고 GitHub 저장소에 업로드합니다.
2. GitHub Pages를 켭니다.
3. 스프레드시트 공유 설정을 `링크가 있는 모든 사용자 보기 가능`으로 설정합니다.
