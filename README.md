# 백솔 업보리스트

Google Spreadsheet와 연동되는 GitHub Pages용 업보리스트입니다.

## 적용 내용
- 빨간색으로 표시된 캐릭터 프로필 영역 제거
- 정렬 드롭다운 제거
- 통계 카드 제거
- 부제목을 `백솔의 달콤살벌 업보 기록소`로 변경
- 업보 항목을 가로 태그형에서 세로 리스트형으로 변경
- 0 또는 빈칸 자동 숨김

## 사용법
1. 이 폴더의 파일을 GitHub 저장소에 업로드합니다.
2. GitHub Pages를 켭니다.
3. 스프레드시트 공유 설정을 `링크가 있는 모든 사용자 보기 가능`으로 설정합니다.
